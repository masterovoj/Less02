Структура тегов

Тег без закрывающего элемента.
К таким тегам относятся теги < img >, < br >, < input > и т.д.

Структура записи следующая:

< имя_тега имя_атрибута="значение_атрибута" имя_атрибута_2="значение_атрибута_2" >

Пример:

< img src="img/image.jpeg" alt="Картинка" >

Тег с закрывающим элементом
К таким тегам относятся теги, которые имеют закрывающия элемент (такой же тег, только перед ним стоит слэш)

Структура записи следующая:

< имя_тега имя_атрибута="значение_атрибута" > . . . </ имя_тега >

Пример:

< H1 > Заголовок 1 </H1>

Структура страницы HTML

Страница HTML имеет определенную структуру. Для этого используются специальные теги, такие как header - (шапка страницы), nav - (навигационная панель), main - (основной раздел страницы, где распологается основной контент и footer - подвал страницы.

Пример структуры страницы:
<!-- Шапка сайта header -->
< header class="header" >

<!-- Навигация сайта - блок nav -->
< nav class="navigation" >
</nav >

</ header >

<!-- Тело сайта main -- >
< main >

< div class="" title="Я подсказка" >

Lorem ipsum dolor sit amet consectetur adipisicing elit.
 Veritatis delectus aspernatur eligendi nobis distinctio
 corporis iure dolore pariatur aut. Perspiciatis reprehenderit
r illum eius quis a libero autem porro officia est.

</ div >

</ main >

<!-- Подвал страницы -- >
< footer >

</ footer >

Строчные теги

Строчные теги - это теги, которые занимают столько места, сколько их размер, а не целиком строку. Строчные теги при написании один за другим, будут располагаться рядом и не выталкивают друг друга на новую строку.

Пример строчных тегов:

<!-- гиперссылка < a > -->
< a tadget="_blank" href="http://google.com" > Текст ссылки (анкор) </ a >

<!-- Тег для выделения чего-либо - < span > -->
< p > Lorem ipsum dolor sit amet < span > consectetur </ span > adipisicing elit. </ p >

Блочные теги

Блочные теги- это теги, которые независимо от своего размера, занимают всю строку полностью на странице. Блочные теги не могут располагаться рядом в одной строке. Они обязательно будут выталкивать друг друга на новую строку.

Пример блочных тегов:

Заголовки H2...H6

Я заголовок 2 уровня

Я заголовок 3 уровня

Я заголовок 4 уровня

Маркированный список < ul >

  • Элемент списка №1
  • Элемент списка №2
  • Элемент списка №3
  • Элемент списка №4

Нумированный список < ol >

  1. Элемент списка №1
  2. Элемент списка №2
  3. Элемент списка №3
  4. Элемент списка №4
